Crystal - Yet another Arch Linux Repository

You've reached the website for my personal Arch Linux repository which distributes packages that are not provided by official repositories.

To use this repository, install and trust the signing key first.

# pacman-key --keyserver hkp://pgp.mit.edu --recv-keys E21E057C731FE0E27BA391E79E81303CA7470095
# pacman-key --lsign-key E21E057C731FE0E27BA391E79E81303CA7470095

Then add the following lines to /etc/pacman.conf:

[crystal]
Server = https://repo.project-ice.org/archlinux/$arch
